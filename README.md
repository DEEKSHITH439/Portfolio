# Portfolio

This is the personal portfolio website of Deekshith K B. It is a modern, responsive portfolio designed to highlight education, projects, internships, skills, and achievements.

## Open locally

1. Open `index.html` in your browser.
2. Or serve the folder using a local web server (for example, `Live Server` in VS Code).

## What’s included

- Hero introduction with profile highlights
- About section with education and strengths
- Skills section with progress visuals
- Featured projects with modal details
- Internship experience and achievement highlights
- Contact form and links
